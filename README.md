# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/chosen-one24/InterVirta.git
```

---

## Install Frontend

```bash
cd frontend
npm install
```

---

## Install Backend

```bash
cd backend
npm install
```

---

## Environment Variables

Create a `.env` file inside the backend.

```env
PORT=

MONGODB_URI=

JWT_SECRET=

GOOGLE_GENAI_API_KEY=
```

---

## Run Backend

```bash
npm run dev
```

---

## Run Frontend

```bash
npm run dev
```
